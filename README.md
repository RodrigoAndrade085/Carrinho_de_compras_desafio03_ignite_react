### :computer: Rodando Projeto:

  Clone o repositório:  
  `https://github.com/RodrigoAndrade085/Carrinho_de_compras_desafio03_ignite_react.git`
  
  Acesse a pasta do projeto:  
  `cd Carrinho_de_compras_desafio03_ignite_react`
  
  Instale as dependências:  
  `yarn ou npm install`

  Rode o server  
  `yarn server`
  
  Rode o front  
  `yarn start`
